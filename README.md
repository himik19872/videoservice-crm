# 🏠 VideoService CRM

Система управления заявками домофонной компании с мобильным приложением для мастеров, GPS-трекингом, складским учётом и уведомлениями через мессенджер Max.

## ✅ Что уже работает

### 📋 Заявки
- Создание заявок (ремонт, подключение, монтаж, ТО, продажа, обследование)
- Назначение мастеров по районам, помощники (helpers)
- Полная статусная модель: новая → назначена → принята → в работе → на паузе / нужна помощь → выполнена → подтверждена → отменена
- История изменений по каждой заявке (кто, когда, с GPS-координатами)
- **Сортировка по 8 полям**: дата, мастер, район, адрес, стоимость, приоритет, номер, клиент
- **Фильтрация**: по статусу, поиск по адресу/клиенту/номеру
- Фото/видео фиксация (требование отчёта, загрузка медиа)
- Учёт оплат: наличные/безналичные, привязка к заявке, сдача в кассу, авто-статус «оплачено»
- Автогенерация номера заявки (ЗАЯВ-ГГММДД-XXXX)

### 👨‍🔧 Мастера и сотрудники
- **14 ролей**: Администратор, Диспетчер, Мастер, Монтажник, Инженер, Главный инженер, Начальник сервисной службы, Технический директор, Исполнительный директор, Генеральный директор, Делопроизводитель, Бухгалтер, Кассир, Секретарь
- **Гибкая система прав**: базовые права по роли + персональные настройки (расширение/отзыв)
- **Единое место управления**: `/settings/admin` — создание, редактирование, удаление, права
- GPS-трекинг: координаты при смене статуса, история на карте, интеграция с Traccar-маяками
- Смены: открытие/закрытие, подсчёт часов
- Распределение по районам обслуживания

### 👥 Клиенты и адреса
- База клиентов с расширенным адресом (город, улица, дом, корпус, квартира, подъезд)
- DaData-автоподстановка адресов при создании
- История заявок клиента
- Привязка к Max (max_user_id), авто-линковка по номеру телефона

### 🏢 Дома и здания
- Обслуживаемые адреса (дома) с типами оборудования
- Расширенная адресация: тип улицы, микрорайон, кол-во квартир и подъездов

### 📦 Склад и оборудование (Inventory)
- Каталог оборудования: домофоны, камеры, вызывные панели, замки, мониторы, кабель, монтажные комплекты
- Серийные номера, модели, закупочные/продажные цены, гарантийные сроки
- **Статусы**: на складе → у мастера → установлено → возвращено → брак → списано
- **Движения**: приход, выдача мастеру, возврат, установка клиенту, списание
- Полная история с автообновлением остатков
- Выдача мастерам под отчёт, отчёт по остаткам у мастеров

### 💰 Финансы
- Учёт оплат: наличные, карта, перевод, онлайн
- Авто-статус «оплачено» при полной сумме
- Расчёт зарплаты мастеров (% от выполненных заявок + премии − удержания)
- Финансовые отчёты за период

### 💬 Max Бот
- Уведомления клиентам о статусе заявок
- Уведомление при назначении мастера (ФИО + телефон)
- Уведомление при подтверждении с фото/видео
- Webhook для линковки клиентов по телефону
- Запрос статуса заявки через бота

### 📊 Отчёты и дашборд
- Статистика главного экрана (заявки, клиенты, оборудование)
- Отчёты по заявкам за период, по мастерам
- Финансовая отчётность

## 👥 Роли и доступы

| Роль | Описание | Ключевые права |
|------|----------|----------------|
| 👑 Администратор | Полный доступ | Всё + настройки + управление пользователями |
| 📋 Диспетчер | Оперативное управление | Заявки, клиенты, мастера, подтверждение |
| 🔧 Мастер | Полевой сотрудник | Свои заявки, GPS, фото/видео, подпись |
| 🏗️ Монтажник | Монтаж оборудования | Свои заявки, фото/видео фиксация |
| 🛠️ Инженер | Техническая поддержка | Заявки «нужна помощь», оборудование |
| ⭐ Главный инженер | Руководство тех. отделом | Заявки, оборудование, склад, назначение |
| 👁️ Начальник сервисной службы | Контроль сервиса | Заявки, подтверждение, мастера, склад |
| 🔬 Технический директор | Тех. руководство | Почти всё, кроме пользователей |
| 💼 Исполнительный директор | Операционное управление | Заявки, финансы, зарплаты, экспорт |
| 🏢 Генеральный директор | Полный доступ | Всё |
| 📁 Делопроизводитель | Документооборот | Заявки, клиенты, отчёты |
| 🧮 Бухгалтер | Финансовый учёт | Оплаты, зарплаты, отчёты, экспорт |
| 💵 Кассир | Приём платежей | Заявки, оплаты, клиенты |
| 📞 Секретарь | Приём заявок | Заявки, клиенты, мастера, склад, отчёты |

> 💡 **Гибкие права**: администратор может расширить или ограничить права любого сотрудника персонально — через `/settings/admin`.

## 🚀 Быстрый старт

### Одна команда (новый сервер)

```bash
git clone https://github.com/himik19872/videoservice-crm.git
cd videoservice-crm
sudo chmod +x install.sh && sudo ./install.sh
```

После установки:
- CRM: **http://ВАШ_IP:3000**
- API: **http://ВАШ_IP:3000/api/**
- Логин: **admin** / **admin123**

### Локальная разработка

```bash
# 1. Backend
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py loaddata regions.json users.json masters.json user_profiles.json
python manage.py runserver

# 2. Frontend (другой терминал)
cd frontend
npm install
npm start

# 3. Прокси (третий терминал)
cd server
npm install
node server.js

# Открой http://localhost:3000
```

## 📡 API

### Аутентификация
| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/auth/login/` | Вход (username, password → token) |
| POST | `/api/auth/refresh/` | Обновить токен |
| GET | `/api/users/me/` | Текущий пользователь |

### Заявки
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/orders/` | Список (фильтры: status, master, region; сортировка: created_at, cost, master, region, address) |
| POST | `/api/orders/` | Создать |
| GET | `/api/orders/{id}/` | Детали + история |
| PATCH | `/api/orders/{id}/` | Обновить статус / назначить мастера |
| POST | `/api/orders/{id}/confirm/` | Подтвердить (диспетчер) |

### Справочники
`/api/clients/` `/api/masters/` `/api/buildings/` `/api/equipment/` `/api/regions/`

### Склад
| Метод | URL | Описание |
|-------|-----|----------|
| GET/POST | `/api/inventory/` | Список / создать оборудование |
| GET/PUT/DELETE | `/api/inventory/{id}/` | Детали / изменить / удалить |
| GET/POST | `/api/inventory-movements/` | История движений |

### Финансы
| Метод | URL | Описание |
|-------|-----|----------|
| GET/POST | `/api/payments/` | Список оплат / принять оплату |
| GET | `/api/master-salaries/` | Расчёт зарплат мастеров |

### Пользователи и права
| Метод | URL | Описание |
|-------|-----|----------|
| GET/POST | `/api/users/` | Список / создать сотрудника |
| GET/PUT/DELETE | `/api/users/{id}/` | Детали / обновить (включая `permissions`) / удалить |

### GPS и смены
`/api/masters/locations/` — координаты всех мастеров
`/api/shifts/start/` `/api/shifts/end/` — открыть/закрыть смену

## 📱 Мобильное приложение (Expo)

```bash
cd mobile
npm install
npx expo start        # Dev
eas build --platform android --profile preview  # APK
```

**Экраны**: авторизация, список заявок (фильтры, быстрая навигация), детали заявки (GPS история, смена статуса, фото/видео, приём оплаты), склад (список + история движений), оплаты, подпись клиента, карта мастеров.

**Возможности**: тёмная тема, офлайн-режим с автосинхронизацией, push-уведомления, смены.

## 🤖 Max Бот

1. Получить токен: [dev.max.ru](https://dev.max.ru)
2. CRM → Системные настройки → Max Бот → вставить токен
3. Webhook URL: `https://ВАШ_IP:3000/api/max/webhook/`

**Цепочка уведомлений:**
```
📋 Назначен мастер → Клиенту: 👨‍🔧 ФИО + телефон мастера
✅ Подтверждено    → Клиенту: Заявка выполнена + 📸 фото/видео
```
